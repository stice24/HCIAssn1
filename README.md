# PennPlan Roadmap

A future planning interface that displays academic year roadmaps with difficulty ratings and credit information.

## Getting Started

### Clone the Repository
```bash
git clone https://github.com/stice24/HCIAssn1.git
cd HCIAssn1
```

## How to Run

### Option 1: Open Directly
Simply open `index.html` in your web browser by double-clicking the file.

### Option 2: Local Server
```bash
python3 -m http.server 8000
```
Then navigate to `http://localhost:8000/index.html`

### Option 3: Live Server (VS Code)
If you have the Live Server extension installed, right-click `index.html` and select "Open with Live Server".

## Features

- 4-year academic roadmap overview
- Color-coded difficulty ratings (green/orange/red)
- Semester-by-semester credit breakdowns
- Responsive mobile-first design
- Clean, modern UI with hover effects
